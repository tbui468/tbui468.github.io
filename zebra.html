About

Motivation

Zebra is a strongly typed scripting language. Programs written in dynamically typed languages are difficult to maintain as they get larger. 1000 lines of C++ (a very complex language, for better or worse) is easier to debug than a 1000 lines of Python. Yes, I know that python has an optional type system, but I feel that it should be required for languages aimed at beginners.

Current Progress

Very much a work in progress. Variables and functions are implemented (along with type checking), but the code isn't robust - there are a lot of edges cases that haven't been tested. Currently working on adding structs to group data. Classes are currently not a priority.

The current implementation runs the program by walking down each branch in the AST (a tree-walk interpreter), which is slow. My future plan is to write a compiler to change the input source code into bytecode, and then run that on a virtual machine.

Getting Started

Building From Source

The source code is available on Github at https://github.com/tbui468/zebra. I'm not distributing executables at the time, so you'll have to build on your own machine. I tested on MSVC and Clang, but it should work fine using GCC. CMake is needed to generate the build system. The following commands will use the default build system on your machine.

git clone https://github.com/tbui468/zebra
cd zebra
mkdir build
cd build
cmake ..
cmake --build .

The executable should be in zebra/build/src/Debug

Code Example

Here's some example code to get you started on the syntax. We compute the 1st, 5th, and 10th fibonacci numbers and initialize a struct of type ThreeNumbers using them. Then we change our minds and replace the second data field with the 6th fibonacci number.

            
                //Declare a struct to hold three integers.
                ThreeInts :: struct {
                    name: string = "default";
                    first: int = 0;
                    second: int = 0;
                    third: int = 0;
                }

                //Declare a function to compute the nth number in the fibonacci sequence
                fib :: (n: int) -> int {
                    if (n < 2) {
                        return n;
                    } else {
                        return fib(n - 1) + fib(n - 2);
                    }
                }

                my_nums: ThreeInts("Fibonacci numbers!", fib(1), fib(5), fib(10));

                print(my_nums);
                
                new_second: int = fib(6);

                my_nums.second = new_second;

                print(my_nums);
Program console output:
"Fibonacci numbers!", 1, 5, 55
"Fibonacci numbers!", 1, 8, 55
