About

Motivation

Zebra is a strongly typed scripting language. Kind of like C without the memory management.

Current Progress

Very much a work in progress. Variables and functions are implemented (along with type checking), but the code isn't robust - there are a lot of edges cases that haven't been tested. Currently working on adding structs to group data. Classes are currently not a priority.


The current implementation runs the program by walking down each branch in the AST (a tree-walk interpreter), which is slow. My future plan is to compile the source text into bytecode, and then run that on a virtual machine.

Getting Started

Building From Source

The source code is available on Github at https://github.com/tbui468/zebra. I'm not distributing executables at the time, so you'll have to build on your own machine. I tested on MSVC and Clang, but it should compile fine using GCC. CMake is needed to generate the build system. The following commands will use the default build system on your machine. The executable should be in zebra/build/src/Debug

                
                    git clone https://github.com/tbui468/zebra
                    cd zebra
                    mkdir build
                    cd build
                    cmake ..
                    cmake --build .

Code Example

Here's some example code to get you started on the syntax. We compute the 1st, 5th, and 10th fibonacci numbers and initialize a struct of type ThreeNumbers using them. Then we change our minds and replace the second data field with the 6th fibonacci number.

                
                    //Define a struct to hold three integers.
                    ThreeInts :: {
                        name: string = "default";
                        first: int = 0;
                        second: int = 0;
                        third: int = 0;
                    }

                    //Define a function to compute the nth number in the fibonacci sequence
                    fib :: (n: int) -> int {
                        if (n < 2) {
                            return n;
                        } else {
                            return fib(n - 1) + fib(n - 2);
                        }
                    }

                    my_nums: ThreeInts("Fibonacci numbers!", fib(1), fib(5), fib(10));

                    print(my_nums);     //"Fibonacci numbers!", 1, 5, 55
                    
                    //I changed my mind - let's replace the second number.
                    my_nums.second = fib(6);

                    print(my_nums);     //"Fibonacci numbers!", 1, 8, 55

Built-In Types

Integer

Declaration:

                
                a: int = 10;

Assignment:

                
                b: int = 0;
                b = 10 * 3;

Supported operators:

                
                c: int = 5;
                d: int = 2;

                //unary
                print(-c);       //-5

                //arithmetic
                print(c + d);    //7
                print(c - d);    //3
                print(c * d);    //10
                print(c / d);    //2
                print(c % d);    //1

                //equality
                print(c == d);   //false
                print(c != d);   //true

                //inequality
                print(c < d);    //false
                print(c <= d);   //false
                print(c > d);    //true
                print(c >= d);   //true

Bool

Declaration:

Custom Types

What's a Struct?

Structs can be used to groups related data together. Structs can be composed of the built-in data types, or other structs.

Declaration

To declare a new struct, follow the name with double colons and then the body. All fields must be set to a default value. 

                
                    Person :: {
                        name: string        = "John";
                        age: int            = 35;
                        hair_color: string  = "Brown";
                        height: float       = 175.1;
                    }

Instantiating and Assignment

New structs can be created in three ways: using default data, using custom data, or copying the data from a different struct instance. Struct instances can also be assigned a different instance, overwriting their data fields.

Note: a custom constructor must have the same number of arguments as datafields in the struct definition, and the order of appearance of the fields in the definition corresponds to the order of the arguments. 

                
                    Dog :: {
                        name: string = "Mittens";
                        age: int = 3;
                    }

                    //creating three dogs
                    my_dog: Dog(); //using default name "Mittens" and age 3
                    other_dog: Dog("Puppers", 2); //with name "Puppers" and age 2
                    copy_dog: Dog(other_dog); //a copy of other_dog (with name "Puppers" and age 2)

                    //assigning my_dog to copy_dog
                    my_dog = copy_dog //now all three dogs have the same name and age

Accessing and Setting Fields

Fields on structs can be accessed and set using dot notation.

                
                    Car :: {
                        color: string = "red";
                        age: int 20;
                    }

                    my_car: Car("blue", 5);
                    print("My care is " + my_car.color + "."); //"My car is blue."

                    my_car.color = "green";
                    print("Then I painted it " + my_car.color + "."); //"Then I painted it green."

Usage

A struct can be used just like any build-in type.

                
                    Cat :: {
                        name: string = "Whiskers";
                        age: int = 2;
                    }

                    print_details :: (cat: Cat) {
                        last_bit: string = " years old.";
                        if (cat.age == 1) {
                            last_bit = " year old."; 
                        }

                        print(cat.name + " is " + string(cat.age) + last_bit);
                    }

                    my_cat: Cat();
                    other_cat: Cat("Bubbles", 1);

                    print_details(my_cat); //"Whiskers is 2 years old."
                    print_details(other_cat): //"Bubbles is 1 year old.";
