About

Motivation

Zebra is a strongly typed scripting language. Kind of like C without the memory management.

Current Progress

Very much a work in progress. Variables and functions are implemented (along with type checking), but the code isn't robust - there are a lot of edges cases that haven't been tested. Currently working on adding structs to group data. Classes are currently not a priority.


The current implementation runs the program by walking down each branch in the AST (a tree-walk interpreter), which is slow. My future plan is to compile the source text into bytecode, and then run that on a virtual machine.

Getting Started

Building From Source

The source code is available on Github at https://github.com/tbui468/zebra. I'm not distributing executables at the time, so you'll have to build on your own machine. I tested on MSVC and Clang, but it should compile fine using GCC. CMake is needed to generate the build system. The following commands will use the default build system on your machine. The executable should be in zebra/build/src/Debug

                
                    git clone https://github.com/tbui468/zebra
                    cd zebra
                    mkdir build
                    cd build
                    cmake ..
                    cmake --build .

Code Example

Here's some example code to get you started on the syntax. We compute the 1st, 5th, and 10th fibonacci numbers and initialize a class called ThreeNumbers to hold them. Then we change our minds and replace the second data field with the 6th fibonacci number.

                
                    //Define a class to hold three integers.
                    ThreeInts :: class {
                        name: string = "default"
                        first: int = 0
                        second: int = 0
                        third: int = 0
                    }

                    //Define a function to compute the nth number in the fibonacci sequence
                    fib :: (n: int) -> int {
                        if n < 2 {
                            return n
                        } else {
                            return fib(n - 1) + fib(n - 2)
                        }
                    }

                    my_nums: ThreeInts("Fibonacci numbers!", fib(1), fib(5), fib(10))

                    print(my_nums)     //"Fibonacci numbers!", 1, 5, 55
                    
                    //I changed my mind - let's replace the second number.
                    my_nums.second = fib(6)

                    print(my_nums)     //"Fibonacci numbers!", 1, 8, 55

Built-In Types

Integers

Declaration:

                
                a: int = 10

Assignment:

                
                b: int = 0
                b = 10 * 3

Supported operators:

                
                c: int = 5
                d: int = 2

                //unary
                print(-c)       //-5

                //arithmetic
                print(c + d)    //7
                print(c - d)    //3
                print(c * d)    //10
                print(c / d)    //2
                print(c % d)    //1

                //equality
                print(c == d)   //false
                print(c != d)   //true

                //inequality
                print(c < d)    //false
                print(c <= d)   //false
                print(c > d)    //true
                print(c >= d)   //true

Floats

Declaration:

                
                a: float = 10.2

Assignment:

                
                b: float = 3.
                b = 10.2 * 3.2

Supported operators:

                
                //Same operators as integers, except there's no modulus operator %.
                //Note: Be careful about using equality (==, !=) and inequalities (<, <=, >, >=) with floating points.

Strings

Declaration:

                
                a: string = "dog"

Assignment:

                
                b: string = "Potato"

                c: string = "Cute"

Booleans

Declaration:

                
                a: bool = true

Assignment:

                
                b: bool = true
                b = false

Supported operators:

Control Flow

If / Else

All control flow statement bodies must be surrounded by braces.

                
                    a: bool = true
                    if a {
                        print("This will print")
                    }

                    b: int = 11
                    if b < 10 {
                        print("This will not print")
                    } else {
                        print("This will print")
                    }

For Loops

C-style syntax. This likely will change in the future. The initializer and update statements can be omitted. The 'break' keyword is not implemented, so be careful with infinite loops.

                
                    for i: int = 0, i < 3, i = i + 1 {
                        print(i)
                    }

                    //This will print:
                    //0
                    //1
                    //2

                    //omitting intializer and update
                    j: int = 5
                    for , j >= 0, {
                        print(i)
                        j = j - 2
                    }

                    //This will print:
                    //5
                    //3
                    //1

While Loops

                
                    while i: int = 0 <= 5 {
                        if i % 2 == 1 {
                            print(i)
                        }
                        i = i + 1
                    }

                    //This will print:
                    //1
                    //3
                    //5

Functions

Declarations

The following snippet declares a functin called 'double'. 'double' has one integer paramter 'n'. The return type is also an integer. The function body returns 2 multiplied by n.

Function 'print_double' has the same parameter, but does not return a value (indicated by leaving the return type blank). The return arrow inside the body of a function with no return is optional. 

                
                    import io

                    //takes one integer as a parameters, and returns an integer
                    double :: (n: int) -> int {
                        -> 2 * n
                    }

                    //takes in one integer as a parameter, and has no return value.
                    print_double :: (n: int) -> {
                        print(2 * n)
                        -> //optional return statement
                    }

                    //takes in two floats as parameters, and has no return value.  
                    early_return :: (n: float, m: float) -> {
                        if n < 5.0 {
                            ->       //will return from function, and skip calling 'print(n * m)' below
                        }

                        print(n * m)
                    }

Calls

Functions are called using the keyword followed by any arguments surrounded by parenthesis 

                
                    import io

                    greeting :: (name: string) -> string {
                        -> "Hello, " + name + "."
                    }

                    print(greeting("Tim"))  //"Hello, Tim."
                    print(greeting("Mittens")) //"Hello, Mittens."

Anonymous Functions

Functions can be declared and immediately passed in as parameters. The function 'sum' takes in an anonymous function as a third parameter and applies it to the first two parameters. 

                
                    import io

                    //function takes three parameters: integers 'a' and 'b', and a function 'fn'
                    //that has one integer parameter and returns an integer
                    sum :: (a: int, b: int, fn: (int) -> int) -> {
                        print(fn(a) + fn(b))
                    }

                    sum(1, 2, (i: int) -> int { -> 2 * i}) //'6'
                    sum(1, 2, (i: int) -> int { -> 10 * i}) //'30'

Classes

Declaration

To declare a new class, follow the name with double colons and then the body. All fields must be set to a default value. 

                
                    Person :: class {
                        name: string        = "John"
                        age: int            = 35
                        hair_color: string  = "Brown"
                        height: float       = 175.1
                    }

Instantiating and Assignment

New classes can be created in three ways: using default data, using custom data, or copying the data from a different struct instance. Class instances can also be assigned a different instance, overwriting their data fields.

Note: a custom constructor must have the same number of arguments as datafields in the class definition, and the order of appearance of the fields in the definition corresponds to the order of the arguments. 

                
                    Dog :: class {
                        name: string = "Mittens"
                        age: int = 3
                    }

                    my_dog: Dog = Dog() //name "Mittens" and age 3
                    copy_dog: Dog = my_dog //a copy of my_dog
                    
                    other_dog: Dog = Dog()
                    other_dog = my_dog //assigning to my_dog

Accessing and Setting Fields

Fields on classes can be accessed and set using dot notation.

                
                    import io

                    Car :: class {
                        color: string = "red"
                        age: int = 20
                    }

                    my_car: Car = Car()
                    print("My care is " + my_car.color + ".") //"My car is red."

                    my_car.color = "green"
                    print("Then I painted it " + my_car.color + ".") //"Then I painted it green."

Class Functions

Class functions can be declared just like any other function. Class functions are called using dot notation. 

                
                    import io

                    Cat :: class {
                        name: string = "Whiskers"
                        age: int = 2

                        speak :: () -> {
                            print(name)
                        }

                        multiply_age :: (n: int) -> int {
                            -> n * age
                        }
                    }

                    my_cat: Cat = Cat()
                    my_cat.speak() //"Whiskers"
                    print(my_cat.multiply_age(5)) //10

Inheritance

Single inheritance is currently supported. Base class fields and methods can be overwritten by the derived class. 

                
                    import io

                    Animal :: class {
                        classification: string = "mammalia" 
                        get_verb :: () -> string {
                            -> "run"
                        }
                    }

                    Deer :: class < Animal {
                    }

                    Shark :: class < Animal {
                        classification: string = "chondrichthyes"
                        get_verb :: () -> string {
                            -> "swim"
                        }
                    }

                    my_deer: Deer = Deer()
                    my_shark: Shark = Shark()

                    print(my_deer.classification) //"mammalia"
                    print(my_deer.get_verb()) //"run"
                    print(my_shark.classification) //"chondrichthyes"
                    print(my_deer.get_verb()) //"swim"
